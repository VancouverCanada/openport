# Support

OpenMCP is stewarded by **Accentrust Inc.** and **Sebastian Zhu**.

## Security issues

Do not open public issues for vulnerabilities.  
Follow `SECURITY.md` for private reporting.

## Usage questions

Open a GitHub issue with:

- OpenMCP version
- Node version
- expected behavior
- actual behavior
- reproducible steps

## Feature requests

Open a GitHub issue using the feature template and include:

- use case
- expected protocol behavior
- compatibility impact
- security considerations

## Response targets

- triage acknowledgement: within 3 business days
- severity and scope decision: within 7 business days
